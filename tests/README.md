# 测试目录说明

## 目录作用
`tests/` 目录用于存放本项目的基础测试文件，帮助开发者检查关键文件和模块是否能够正常使用。

## 当前已有测试
目前测试目录中已包含基础导入测试，例如：
- `test_imports.py`：用于检查项目基础模块是否能够正常导入，以及项目根目录是否存在。

## 运行方式
在项目根目录下执行：

```bash
pytest

如果只运行测试目录下的测试，可以执行：

```bash
pytest tests

## 运行环境

建议在已安装项目依赖的 Python 环境中运行测试。

## 结果说明

如果测试通过，终端会显示通过的测试数量；
如果测试失败，终端会显示失败的测试项和对应报错信息。
