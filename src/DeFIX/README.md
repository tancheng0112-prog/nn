## 摘要
在城市环境中安全行驶且不违反任何交通规则，是可靠自动驾驶的核心性能指标。
本文提出一种基于强化学习（RL）的方法，通过提取违规点位，并在这些区域构建微型场景来训练强化学习智能体，从而检测并修复（DETECT & FIX，DeFIX）模仿学习（IL）智能体的故障。
DeFIX 是一个持续学习框架：故障场景提取与强化学习训练在无限循环中执行。每训练一个新策略并加入策略库后，策略分类器会在评估阶段决定激活哪一个策略。
实验证明：即使只使用一个在模仿学习故障场景上训练的强化学习智能体，DeFIX 也能达到或超越当前最先进的模仿学习与强化学习自动驾驶方法。
我们在 CARLA 模拟器中最具挑战性的地图（Town05）上训练并验证了方法，该地图包含复杂、真实且对抗性的驾驶场景。

## 方法概述
- 阶段 1：通过规则-based CARLA 自动驾驶采集初始数据集，训练模仿学习智能体，并使用 DAgger 方法持续优化。
- 阶段 2：通过评估 DAgger 训练的模仿学习策略，在违规/故障位置构建微型场景。在该阶段的每个循环中，一个不同的强化学习智能体在一个微型场景上训练，并加入强化学习策略库。
为了在测试时激活模仿学习智能体或某一个强化学习智能体，本文使用监督学习训练了一个策略分类网络。

## 网络结构
所有网络均使用预训练 ResNet-50 作为图像特征提取主干。ResNet-50 的最后一层被重新初始化，并在训练刹车分类器与策略分类器时从头训练。
系统从传感器获取：
- 速度
- 朝向
- 位置
- 前视相机图像
目标路径点通过 IMU 和 GPS 转换到局部相对坐标系。
