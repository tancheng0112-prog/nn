# CARLA恶劣天气鲁棒性测试与自适应感知系统

基于CARLA 0.9.14，在7种天气场景（晴朗/多云/小雨/暴雨/浓雾/夜间/夜间暴雨）下测试感知系统可靠性，并根据图像质量动态切换相机/LiDAR融合权重。

## 核心功能

- **天气场景引擎**：自动切换7种天气，支持渐变过渡
- **图像质量评估**：模糊度/亮度/可见度三维打分
- **LiDAR自适应感知**：DBSCAN点云聚类检测障碍物
- **自适应融合**：晴天相机主导(0.8)，恶劣天气LiDAR主导(0.7)
- **鲁棒性评分**：碰撞率+图像质量保持+切换合理性，输出0-100分报告

## 快速开始

```bash
pip install -r requirements.txt
python weather_robustness.py
```

需要先启动CARLA 0.9.14服务端。
